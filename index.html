¿Por qué todavía desarrollamos para mútiples plataformas?

Desventajas de lo nativo

  • Competencia en cada plataforma requerida
  • Bases de código completamente separadas
  • Desarrollo oportuno y costoso
  • Rendimientos decrecientes
“¿Existe alguna alternativa?”

Apps Hibrídas!

  • HTML5 que actúa como nativo
  • Web envuelta en capa nativa
  • Acceso directo a API nativas
  • Cordova/Phonegap
  • Entorno de desarrollo web familiar

Ya no es el 2007....

¡Los dispositivos móviles han mejorado rápidamente!

¿Entonces, qué necesitamos?

  • Un Framework Open Source
  • Ayuda a cerrar la brecha entre la web y los nativos/li>
  • Componentes e interacciones UI nativos y ricos en estilos
  • UI API, no solo widgets en jQuery
  • Una comunidad dedicada, impulsada por el espíritu
